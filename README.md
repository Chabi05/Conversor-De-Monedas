# 🚀 Conversor de Monedas en C

Este proyecto es una aplicación de consola desarrollada en **C** que permite realizar conversiones entre Quetzales (GTQ), Dólares (USD) y Euros (EUR) utilizando tasas de cambio actuales.

## 📝 Descripción

Un programa interactivo que utiliza funciones y estructuras `switch` para gestionar un menú de conversión de divisas. Es ideal para practicar la lógica de flujos y operaciones aritméticas en lenguaje C.

## 💻 Entorno y Tecnologías

* **Lenguaje:** C (GCC)
* **Editor:** Visual Studio Code
* **Sistema:** Ubuntu (VirtualBox)

## 🛠️ Instalación

Para compilar programas en C dentro de tu Ubuntu, asegúrate de tener instalado el compilador `gcc`. Si no lo tienes, instálalo con:

sudo apt update && sudo apt install build-essential

🏃 Pasos para la Ejecución
📂 Abre Visual Studio Code en tu Ubuntu.

📄 Crea el archivo conversor.c.

💾 Pega el codigo que aparece en este repositorio

🖥️ En la terminal, compila el programa:
gcc conversor.c -o conversor
🚀 Ahora, ejecútalo:
./conversor
